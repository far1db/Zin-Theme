Zin Theme

Zin Theme is a modern and neat HTML Template. It can be used for multiple types of websites. It is perfect for a portfolio or a blog.

Get now See demos

Responsive

This template is fully responsive, thus mobile friendly. Giving a full experience for mobile users is a fundamental feature of a modern website. Therefore, Zin Theme was built with respect for mobile devices standards and requirements.

User friendly

Zin Theme was design with user comfort in mind. From colors to usability, each aspect of this template has been thaught to provide the best experience possible to the user. Featuring a modern and neat design, Zin Theme is attractive and captivates the user and direct it to the content.

Customizable

Zin Theme is a fully customizable template. A detailed documentation is included with the template to help you use the multiple features available for you. The CSS files are clearly written and you can add you own touch to the theme if you like. You also have the PSDs files which may help you take the customization much further.

One/Multi page

This template can be used to create either a single page website or a more complexe website using multiple pages with the example pages included.
